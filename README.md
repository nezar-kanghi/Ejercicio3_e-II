En este ejercicio se nos pide realizar un programa que calcula clasificacion y estadisticas usando consultas SQL con JOIN y funciones de agregacion. Empezamos generando un menu mediante do while y switch, en el primer case llamaremos al metodo clasificacionGeneral, este metodo realizara una consulta que nos dara todos los datos incluida la suma de puntos totales del ciclista y luego nos mostrara los datos, el segundo case llamara al metodo clasificacionEquipos que realizara una consulta que nos dara los datos de los equipos, su pais puntos totales, nombre... al igual que el primer motodo nos motrara los datos por pantalla mediante ResultSet-executeQuery. El ultimo metodo seria rankingEtapas este metodo empezara realizando una consulta que nos dara el numero, origen, destino, distancia_km y fecha la cual solo mostrara las 3 primeras filas mediante FETCH FIRST 3 ROWS ONLY asi generaremos el top 3, acabaremos mostrando los datos.
